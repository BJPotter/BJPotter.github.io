иҮӘеўһиҮӘеҮҸиҝҗз®—з¬Ұ

з¬ҰеҸ·еңЁеүҚе°ұе…ҲеҠ /еҮҸпјҢз¬ҰеҸ·еңЁеҗҺе°ұеҗҺеҠ /еҮҸ

дҪҚиҝҗз®—з¬Ұ

  1. жҢүдҪҚдёҺ

еҜ№дёӨдёӘж“ҚдҪңж•°зҡ„жҜҸдёҖдҪҚжү§иЎҢйҖ»иҫ‘дёҺж“ҚдҪңпјҢеҸӘжңүеҪ“дёӨдёӘж“ҚдҪңж•°еҜ№еә”дҪҚйғҪдёә1ж—¶пјҢз»“жһңдҪҚжүҚдёә1пјҢеҗҰеҲҷдёә0гҖӮ

  1. жҢүдҪҚжҲ–

еҜ№дёӨдёӘж“ҚдҪңж•°зҡ„жҜҸдёҖдҪҚжү§иЎҢйҖ»иҫ‘жҲ–ж“ҚдҪңпјҢеҸӘиҰҒдёӨдёӘж“ҚдҪңж•°еҜ№еә”дҪҚдёӯжңүдёҖдёӘдёә1пјҢз»“жһңдҪҚе°ұдёә1пјҢеҗҰеҲҷдёә0гҖӮ

  1. жҢүдҪҚејӮжҲ–

еҜ№дёӨдёӘж“ҚдҪңж•°зҡ„жҜҸдёҖдҪҚжү§иЎҢйҖ»иҫ‘ејӮжҲ–ж“ҚдҪңпјҢеҪ“дёӨдёӘж“ҚдҪңж•°еҜ№еә”дҪҚдёҚзӣёеҗҢж—¶з»“жһңдҪҚдёә1пјҢзӣёеҗҢж—¶з»“жһңдҪҚдёә0гҖӮ

  1. жҢүдҪҚйқһ

еҜ№ж“ҚдҪңж•°зҡ„жҜҸдёӘдҪҚжү§иЎҢйҖ»иҫ‘йқһж“ҚдҪңпјҢеҚі0еҸҳдёә1пјҢ1еҸҳдёә0гҖӮ

  1. е·Ұз§»

е°Ҷж“ҚдҪңж•°зҡ„жүҖжңүдҪҚеҗ‘е·Ұз§»еҠЁжҢҮе®ҡзҡ„дҪҚж•°пјҢеҸідҫ§з©әеҮәзҡ„дҪҚз”Ё0еЎ«е……гҖӮx << 1,зӣёеҪ“дәҺ x д№ҳд»Ҙ 2

  1. еёҰз¬ҰеҸ·еҸіз§»

е°Ҷж“ҚдҪңж•°зҡ„жүҖжңүдҪҚеҗ‘еҸіз§»еҠЁжҢҮе®ҡзҡ„дҪҚж•°пјҢе·Ұдҫ§з©әеҮәзҡ„дҪҚз”ЁеҺҹжқҘзҡ„жңҖй«ҳдҪҚеЎ«е……гҖӮеҜ№дәҺжңүз¬ҰеҸ·ж•ҙж•°пјҢз¬ҰеҸ·дҪҚдјҡдҝқжҢҒдёҚеҸҳгҖӮx >> 1,зӣёеҪ“дәҺ x йҷӨд»Ҙ 2гҖӮ

  1. ж— з¬ҰеҸ·еҸіз§»

е°Ҷж“ҚдҪңж•°зҡ„жүҖжңүдҪҚеҗ‘еҸіз§»еҠЁжҢҮе®ҡзҡ„дҪҚж•°пјҢе·Ұдҫ§з©әеҮәзҡ„дҪҚз”Ё0еЎ«е……гҖӮеҜ№дәҺж— з¬ҰеҸ·ж•ҙж•°пјҢз¬ҰеҸ·дҪҚд№ҹдјҡз§»еҠЁгҖӮx >>> 1,зӣёеҪ“дәҺ x йҷӨд»Ҙ 2гҖӮ

жіЁж„ҸпјҡеҪ“дёӨдёӘзӣёеҗҢзҡ„ж•°еҒҡжҢүдҪҚдёҺе’ҢжҢүдҪҚжҲ–ж“ҚдҪңж—¶пјҢз»“жһңйғҪе°ҶжҳҜиҝҷдёӘзӣёеҗҢзҡ„ж•°жң¬иә«гҖӮеҪ“дёӨдёӘзӣёеҗҢзҡ„ж•°иҝӣиЎҢжҢүдҪҚејӮжҲ–ж“ҚдҪңж—¶пјҢз»“жһңжҳҜ0пјӣиҖҢеҪ“дёҖдёӘж•°иҝӣиЎҢжҢүдҪҚйқһж“ҚдҪңж—¶пјҢз»“жһңжҳҜиҜҘж•°зҡ„жҜҸдёӘдҪҚеҸ–еҸҚгҖӮ

continueгҖҒbreakгҖҒreturnзҡ„еҢәеҲ«

continue:з»“жқҹжң¬ж¬ЎеҫӘзҺҜпјҢз»§з»ӯжү§иЎҢдёӢж¬ЎеҫӘзҺҜ

breakпјҡз»“жқҹеҫӘзҺҜ

return: з»“жқҹиҝҗиЎҢзЁӢеәҸ

еҺҹз Ғе’ҢеҸҚз Ғе’ҢиЎҘз Ғ

ж•°еӯ—жҳҜд»ҘиЎҘз Ғзҡ„еҪўејҸеӯҳеңЁи®Ўз®—жңәдёӯ

еҺҹз Ғпјҡж•°еӯ—дәҢиҝӣеҲ¶дҪҚзҡ„жңҖй«ҳдҪҚдёәз¬ҰеҸ·дҪҚпјҢ0дёәжӯЈж•°пјҢ1дёәиҙҹж•°пјҢе…¶дҪҷдҪҚиЎЁзӨәж•°еӯ—зҡ„еҖјгҖӮ

еҸҚз ҒпјҡжӯЈж•°зҡ„еҸҚз Ғе’ҢеҺҹз ҒжҳҜдёҖиҮҙзҡ„пјҢиҙҹж•°зҡ„еҸҚз ҒжҳҜйҷӨз¬ҰеҸ·дҪҚе…¶дҪҷдҪҚеҸ–еҸҚгҖӮ

иЎҘз ҒпјҡжӯЈж•°зҡ„иЎҘз Ғе’ҢеҺҹз ҒжҳҜдёҖиҮҙзҡ„пјҢиҙҹж•°зҡ„иЎҘз ҒжҳҜеңЁеҸҚз Ғзҡ„еҹәзЎҖдёҠеҠ 1гҖӮ

ж•°жҚ®зұ»еһӢ

еҹәжң¬ж•°жҚ®зұ»еһӢ

жө®зӮ№ж•°еңЁи®Ўз®—жңәдёӯзҡ„еӯҳеӮЁеҲҶдёәдёүйғЁеҲҶеҲҶеҲ«жҳҜз¬ҰеҸ·дҪҚпјҲеҢәеҲҶжӯЈиҙҹж•°пјҢ0жҳҜж•ҙж•°пјүгҖҒжҢҮж•°пјҲе№Ӯж¬ЎзЎ®е®ҡж•ҙж•°зҡ„иҢғеӣҙпјүгҖҒе°Ҹж•°пјҲе°Ҹж•°дҪҚзЎ®е®ҡдәҶзІҫеәҰпјү

жҜ”еҰӮfloatзұ»еһӢжҳҜ32дҪҚпјҢжҳҜеҚ•зІҫеәҰжө®зӮ№иЎЁзӨәжі•пјҡз¬ҰеҸ·дҪҚпјҲsignпјүеҚ з”Ё1дҪҚпјҢз”ЁжқҘиЎЁзӨәжӯЈиҙҹж•°пјҢжҢҮж•°дҪҚпјҲexponentпјүеҚ з”Ё8дҪҚпјҢз”ЁжқҘиЎЁзӨәжҢҮж•°пјҢе°Ҹж•°дҪҚпјҲfractionпјүеҚ з”Ё23дҪҚпјҢз”ЁжқҘиЎЁзӨәе°Ҹж•°пјҢдёҚи¶ідҪҚж•°иЎҘ0гҖӮ

иҖҢdoubleзұ»еһӢжҳҜ64дҪҚпјҢжҳҜеҸҢзІҫеәҰжө®зӮ№иЎЁзӨәжі•пјҡз¬ҰеҸ·дҪҚеҚ з”Ё1дҪҚпјҢжҢҮж•°дҪҚеҚ з”Ё11дҪҚпјҢе°Ҹж•°дҪҚеҚ з”Ё52дҪҚгҖӮ

еҸҜд»ҘзңӢеҲ°пјҢеғҸ byteгҖҒshortгҖҒintгҖҒlongиғҪиЎЁзӨәзҡ„жңҖеӨ§жӯЈж•°йғҪеҮҸ 1 дәҶгҖӮиҝҷжҳҜдёәд»Җд№Ҳе‘ўпјҹ

еҜ№дәҺ Java дёӯзҡ„ж•ҙж•°зұ»еһӢпјҲеҰӮ byteгҖҒshortгҖҒintгҖҒlongпјүпјҢе…¶дәҢиҝӣеҲ¶иЎЁзӨәдёӯжңҖй«ҳдҪҚжҳҜз¬ҰеҸ·дҪҚпјҢ0 иЎЁзӨәжӯЈж•°пјҢ1 иЎЁзӨәиҙҹж•°гҖӮеӣ жӯӨпјҢеҰӮжһңиҰҒиЎЁзӨәжңҖеӨ§зҡ„жӯЈж•°пјҢйңҖиҰҒе°ҶйҷӨдәҶжңҖй«ҳдҪҚд№ӢеӨ–зҡ„жүҖжңүдҪҚйғҪи®ҫдёә 1гҖӮдҪҶжҳҜз”ұдәҺжңҖй«ҳдҪҚе·Із»Ҹиў«з”ЁдәҺиЎЁзӨәз¬ҰеҸ·пјҢжүҖд»ҘеңЁиЎЁзӨәжӯЈж•°ж—¶пјҢжңҖеӨ§еҖјйңҖиҰҒеҮҸеҺ»дёҖдёӘеҚ•дҪҚпјҢд»ҘзЎ®дҝқдёҚдјҡжәўеҮәеҸҳжҲҗиҙҹж•°гҖӮ

еҹәжң¬зұ»еһӢе’ҢеҢ…иЈ…зұ»еһӢзҡ„еҢәеҲ«

з”ЁйҖ”пјҡж–№жі•еҸӮж•°е’ҢеҸҳйҮҸзұ»еһӢдёҖиҲ¬з”ЁеҢ…иЈ…зұ»еһӢпјҢеҫҲе°‘з”Ёеҹәжң¬зұ»еһӢ

еӯҳеӮЁж–№ејҸпјҡеҹәжң¬зұ»еһӢзҡ„жҲҗе‘ҳеҸҳйҮҸеӯҳеӮЁеңЁе ҶдёҠпјҢеұҖйғЁеҸҳйҮҸеӯҳеӮЁеңЁж ҲдёҠпјҢеҢ…иЈ…зұ»еһӢйғҪеңЁе ҶдёҠ

еҚ з”Ёз©әй—ҙпјҡеҹәжң¬ж•°жҚ®зұ»еһӢеҚ з”Ёзҡ„з©әй—ҙе°Ҹ

й»ҳи®ӨеҖјпјҡеҢ…иЈ…зұ»еһӢзҡ„й»ҳи®ӨеҖјжҳҜnull,еҹәжң¬зұ»еһӢзҡ„жңүиҮӘе·ұй»ҳи®ӨеҖјпјҢдёҚжҳҜnull

жҜ”иҫғж–№ејҸпјҡеҢ…иЈ…зұ»еһӢдҪҝз”Ё == жҜ”иҫғзҡ„жҳҜеҶ…еӯҳең°еқҖпјҢеҹәжң¬зұ»еһӢ == жҜ”иҫғзҡ„жҳҜеӨ§е°Ҹ

жө®зӮ№иҝҗз®—пјҢзІҫеәҰдёўеӨұй—®йўҳ

и®Ўз®—жңәжҳҜдәҢиҝӣеҲ¶зҡ„пјҢиҖҢдё”и®Ўз®—жңәеңЁиЎЁзӨәдёҖдёӘж•°еӯ—ж—¶пјҢе®ҪеәҰжҳҜжңүйҷҗзҡ„пјҢж— йҷҗеҫӘзҺҜзҡ„е°Ҹж•°еӯҳеӮЁеңЁи®Ўз®—жңәж—¶пјҢеҸӘиғҪиў«жҲӘж–ӯпјҢжүҖд»Ҙе°ұдјҡеҜјиҮҙе°Ҹж•°зІҫеәҰеҸ‘з”ҹжҚҹеӨұзҡ„жғ…еҶөгҖӮиҝҷд№ҹе°ұжҳҜи§ЈйҮҠдәҶдёәд»Җд№Ҳжө®зӮ№ж•°жІЎжңүеҠһжі•з”ЁдәҢиҝӣеҲ¶зІҫзЎ®иЎЁзӨәгҖӮ

еҰӮдҪ•и§ЈеҶіпјҡBigDecimal еҸҜд»Ҙе®һзҺ°еҜ№жө®зӮ№ж•°зҡ„иҝҗз®—пјҢдёҚдјҡйҖ жҲҗзІҫеәҰдёўеӨұгҖӮйҖҡеёёжғ…еҶөдёӢпјҢеӨ§йғЁеҲҶйңҖиҰҒжө®зӮ№ж•°зІҫзЎ®иҝҗз®—з»“жһңзҡ„дёҡеҠЎеңәжҷҜпјҲжҜ”еҰӮж¶үеҸҠеҲ°й’ұзҡ„еңәжҷҜпјүйғҪжҳҜйҖҡиҝҮ BigDecimal жқҘеҒҡзҡ„гҖӮ

и¶…иҝҮlongж•ҙеһӢзҡ„ж•°жҚ®еә”иҜҘеҰӮдҪ•иЎЁзӨә

жіЁж„Ҹпјҡи¶…иҝҮlongд»ҘеҗҺдјҡд»ҺеҜјиҮҙжәўеҮәпјҢдјҡд»ҺжңҖе°ҸеҖјејҖе§Ӣз®—

// и¶…иҝҮиҢғеӣҙдјҡеҸҳжҲҗжңҖе°ҸеҖјејҖе§Ӣи®Ўз®—
 public static void main(String[] args) {
        long l = Long.MAX_VALUE;
        System.out.println(l + 1); // -9223372036854775808
        System.out.println(l + 1 == Long.MIN_VALUE); // true
        System.out.println(l + 100); //-9223372036854775709
        System.out.println(l + 100 == Long.MIN_VALUE); // false
        System.out.println(l + 10000); //-9223372036854765809
        System.out.println(l + 10000 == Long.MIN_VALUE); // false
    }

еҸҳйҮҸ

жҲҗе‘ҳеҸҳйҮҸе’ҢеұҖйғЁеҸҳйҮҸзҡ„еҢәеҲ«

и®ҝй—®дҝ®йҘ°з¬ҰпјҡжҲҗе‘ҳеҸҳйҮҸеҸҜд»Ҙиў« public,private,static зӯүдҝ®йҘ°з¬ҰжүҖдҝ®йҘ°пјҢиҖҢеұҖйғЁеҸҳйҮҸдёҚиғҪиў«и®ҝй—®жҺ§еҲ¶дҝ®йҘ°з¬ҰеҸҠ static жүҖдҝ®йҘ°пјӣдҪҶжҳҜпјҢжҲҗе‘ҳеҸҳйҮҸе’ҢеұҖйғЁеҸҳйҮҸйғҪиғҪиў« final жүҖдҝ®йҘ°гҖӮ

дҪңз”ЁеҹҹпјҡжҲҗе‘ҳеҸҳйҮҸзҡ„дҪңз”ЁеҹҹжҳҜж•ҙдёӘзұ»пјҢеұҖйғЁеҸҳйҮҸдҪңз”ЁеҹҹжҳҜж–№жі•дёӯ

еӯҳеӮЁж–№ејҸпјҡжҲҗе‘ҳеҸҳйҮҸеңЁе ҶеҶ…еӯҳдёӯйҡҸеҜ№иұЎж¶ҲеӨұиҖҢж¶ҲеӨұпјҢеұҖйғЁеҸҳйҮҸеңЁж ҲеҶ…еӯҳдёӯйҡҸж–№жі•жү§иЎҢз»“жқҹж¶ҲеӨұ

й»ҳи®ӨеҖјпјҡжҲҗе‘ҳеҸҳйҮҸеҰӮжһңжІЎжңүиў«иөӢеҲқе§ӢеҖјпјҢеҲҷдјҡиҮӘеҠЁд»Ҙзұ»еһӢзҡ„й»ҳи®ӨеҖјиҖҢиөӢеҖјпјҲдёҖз§Қжғ…еҶөдҫӢеӨ–:иў« final дҝ®йҘ°зҡ„жҲҗе‘ҳеҸҳйҮҸд№ҹеҝ…йЎ»жҳҫејҸең°иөӢеҖјпјүпјҢиҖҢеұҖйғЁеҸҳйҮҸеҲҷдёҚдјҡиҮӘеҠЁиөӢеҖјпјҢеңЁдҪҝз”Ёд№ӢеүҚжҳҫејҸең°еҲқе§ӢеҢ–гҖӮ

йқҷжҖҒеҸҳйҮҸзҡ„дҪңз”Ё

йқҷжҖҒеҸҳйҮҸд№ҹе°ұжҳҜиў« static е…ій”®еӯ—дҝ®йҘ°зҡ„еҸҳйҮҸгҖӮе®ғеҸҜд»Ҙиў«зұ»зҡ„жүҖжңүе®һдҫӢе…ұдә«пјҢж— и®әдёҖдёӘзұ»еҲӣе»әдәҶеӨҡе°‘дёӘеҜ№иұЎпјҢе®ғд»¬йғҪе…ұдә«еҗҢдёҖд»ҪйқҷжҖҒеҸҳйҮҸгҖӮд№ҹе°ұжҳҜиҜҙпјҢйқҷжҖҒеҸҳйҮҸеҸӘдјҡиў«еҲҶй…ҚдёҖж¬ЎеҶ…еӯҳпјҢеҚідҪҝеҲӣе»әеӨҡдёӘеҜ№иұЎпјҢиҝҷж ·еҸҜд»ҘиҠӮзңҒеҶ…еӯҳгҖӮ

еӯ—з¬ҰеһӢе’Ңеӯ—з¬ҰдёІеҸҳйҮҸзҡ„еҢәеҲ«

еҪўејҸ : еӯ—з¬ҰеёёйҮҸжҳҜеҚ•еј•еҸ·еј•иө·зҡ„дёҖдёӘеӯ—з¬ҰпјҢеӯ—з¬ҰдёІеёёйҮҸжҳҜеҸҢеј•еҸ·еј•иө·зҡ„ 0 дёӘжҲ–иӢҘе№ІдёӘеӯ—з¬ҰгҖӮ

еҗ«д№ү : еӯ—з¬ҰеёёйҮҸзӣёеҪ“дәҺдёҖдёӘж•ҙеһӢеҖј( ASCII еҖј),еҸҜд»ҘеҸӮеҠ иЎЁиҫҫејҸиҝҗз®—; еӯ—з¬ҰдёІеёёйҮҸд»ЈиЎЁдёҖдёӘең°еқҖеҖј(иҜҘеӯ—з¬ҰдёІеңЁеҶ…еӯҳдёӯеӯҳж”ҫдҪҚзҪ®)гҖӮ

еҚ еҶ…еӯҳеӨ§е°Ҹпјҡеӯ—з¬ҰеёёйҮҸеҸӘеҚ  2 дёӘеӯ—иҠӮ; еӯ—з¬ҰдёІеёёйҮҸеҚ иӢҘе№ІдёӘеӯ—иҠӮгҖӮ

ж–№жі•

йқҷжҖҒж–№жі•еҸҜд»Ҙи°ғз”ЁйқһйқҷжҖҒж–№жі•еҗ—

йқҷжҖҒж–№жі•жҳҜеұһдәҺзұ»зҡ„пјҢеңЁзұ»еҠ иҪҪзҡ„ж—¶еҖҷе°ұдјҡеҲҶй…ҚеҶ…еӯҳпјҢеҸҜд»ҘйҖҡиҝҮзұ»еҗҚзӣҙжҺҘи®ҝй—®гҖӮиҖҢйқһйқҷжҖҒжҲҗе‘ҳеұһдәҺе®һдҫӢеҜ№иұЎпјҢеҸӘжңүеңЁеҜ№иұЎе®һдҫӢеҢ–д№ӢеҗҺжүҚеӯҳеңЁпјҢйңҖиҰҒйҖҡиҝҮзұ»зҡ„е®һдҫӢеҜ№иұЎеҺ»и®ҝй—®гҖӮ

еңЁзұ»зҡ„йқһйқҷжҖҒжҲҗе‘ҳдёҚеӯҳеңЁзҡ„ж—¶еҖҷйқҷжҖҒж–№жі•е°ұе·Із»ҸеӯҳеңЁдәҶпјҢжӯӨж—¶и°ғз”ЁеңЁеҶ…еӯҳдёӯиҝҳдёҚеӯҳеңЁзҡ„йқһйқҷжҖҒжҲҗе‘ҳпјҢеұһдәҺйқһжі•ж“ҚдҪңгҖӮ

йқҷжҖҒж–№жі•е’Ңе®һдҫӢж–№жі•зҡ„еҢәеҲ«

и°ғз”Ёж–№ејҸпјҡйқҷжҖҒж–№жі•пјҢзӣҙжҺҘзұ»еҗҚ.йқҷжҖҒж–№жі•еҗҚпјҢе®һдҫӢж–№жі•пјҢеҝ…йЎ»е…ҲеҲӣе»әеҜ№иұЎпјҢеҜ№иұЎ.е®һдҫӢж–№жі•

и®ҝй—®йҷҗеҲ¶пјҡйқҷжҖҒж–№жі•еҸӘеҸҜд»Ҙи®ҝй—®йқҷжҖҒеҸҳйҮҸжҲ–иҖ…йқҷжҖҒж–№жі•пјҢдёҚиғҪи®ҝй—®е®һдҫӢеҸҳйҮҸе’Ңе®һдҫӢж–№жі•пјҢе®һдҫӢж–№жі•жІЎжңүиҝҷдёӘйҷҗеҲ¶

еҸҜеҸҳй•ҝеҸӮж•°

Java ж”ҜжҢҒе®ҡд№үеҸҜеҸҳй•ҝеҸӮж•°пјҢжүҖи°“еҸҜеҸҳй•ҝеҸӮж•°е°ұжҳҜе…Ғи®ёеңЁи°ғз”Ёж–№жі•ж—¶дј е…ҘдёҚе®ҡй•ҝеәҰзҡ„еҸӮж•°гҖӮе°ұжҜ”еҰӮдёӢйқўзҡ„иҝҷдёӘ printVariable ж–№жі•е°ұеҸҜд»ҘжҺҘеҸ— 0 дёӘжҲ–иҖ…еӨҡдёӘеҸӮж•°гҖӮеҸҜеҸҳеҸӮж•°еҸӘиғҪдҪңдёәеҮҪж•°зҡ„жңҖеҗҺдёҖдёӘеҸӮж•°пјҢдҪҶе…¶еүҚйқўеҸҜд»Ҙжңүд№ҹеҸҜд»ҘжІЎжңүд»»дҪ•е…¶д»–еҸӮж•°гҖӮйҒҮеҲ°ж–№жі•йҮҚиҪҪзҡ„жғ…еҶөдјҡдјҳе…ҲеҢ№й…Қеӣәе®ҡеҸӮж•°
